# 🚀 Maintenance Data Pipeline

Projeto de engenharia de dados para automação de coleta, processamento e análise de dados de manutenção de ativos.

---

## 📊 Objetivo

Construir um pipeline de dados fim a fim para:
- Extrair dados simulados de manutenção
- Tratar e transformar dados com Python
- Gerar KPIs operacionais
- Criar outputs prontos para BI (Power BI)
- Automatizar todo o fluxo

---

## 🧱 Arquitetura

Data Source → Python ETL → Data Lake (CSV/Excel) → Analytics / BI

---

## ⚙️ Stack utilizada

- Python (Pandas, Matplotlib)
- ETL Pipeline
- Data Analysis
- Data Visualization

---

## 📦 Funcionalidades

- Geração de dados simulados de manutenção
- Pipeline ETL automatizado
- Tratamento e limpeza de dados
- Cálculo de KPIs mensais
- Exportação CSV e Excel
- Geração de gráfico automático
- Sistema de logging

---

## 📈 KPIs gerados

- Tempo parado por período
- Custo de manutenção
- Consolidação mensal de falhas

---

## ▶️ Como executar

pip install -r requirements.txt  
python src/pipeline.py  

---

## 📁 Estrutura do projeto

- src/ → código do pipeline  
- data/ → dados gerados  
- config/ → configurações  
- logs/ → logs de execução  
- plots/ → gráficos  

---

## 📊 Exemplo de saída

- KPI mensal consolidado  
- Gráfico automático de tempo parado  
- Exportação em CSV e Excel

## 🚀 Destaques técnicos

- Automação de fluxo de dados

- Geração de indicadores (KPIs)  
- Pronto para BI (Power BI)
